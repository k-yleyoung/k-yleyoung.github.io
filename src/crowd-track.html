About Crowd Track

Crowd Track is a full-stack iOS app developed using SwiftUI that helps users determine how busy certain locations are in the present and future based on real-time and historical data using interactive graphs.

Currently in beta-testing.

Features

  • Real-time data management and analytics with Firebase.
  • Interactive graphs to visualize data.
  • Historical data analysis to predict future crowd levels.

Technologies Used

  • SwiftUI for front-end development.
  • Firebase for backend and real-time data management.
  • Graphing libraries for interactive data visualization.
